# Capstone-Project-CS3299


### Description

This project focuses on designing an efficient image compression and transmission system using quantization techniques. The goal is to reduce image size while preserving essential visual information, making it suitable for low-bandwidth or resource-constrained environments.

The system converts images into lower bit-depth representations through adaptive quantization, significantly reducing data size. The compressed data is then transmitted and reconstructed at the receiver side with minimal quality loss.

Performance is evaluated using metrics such as PSNR and SSIM to measure the trade-off between compression efficiency and image quality. This approach enables faster and more reliable image transfer in scenarios where bandwidth and power are limited.
